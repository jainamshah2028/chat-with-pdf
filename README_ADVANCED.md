# 🚀 Advanced PDF Chat App

**The most sophisticated PDF chat experience with AI-powered analytics and modern UI!**

## ✨ Quick Start

```bash
# Install dependencies
pip install -r requirements_advanced.txt

# Run the advanced app
streamlit run app_advanced.py
```
