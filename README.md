
# 📄 Chat With Your PDF

An AI-powered app where you can upload any PDF and ask questions about its content using OpenAI + LangChain.

## 🚀 Features
- Upload your PDF
- Ask any question about the content
- Get contextual answers from GPT-3.5 or GPT-4
- Built using LangChain, FAISS, and Streamlit

## 🛠️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🔐 API Key Setup
Add your OpenAI API key in `app.py`:
```python
os.environ["OPENAI_API_KEY"] = "sk-..."
```
